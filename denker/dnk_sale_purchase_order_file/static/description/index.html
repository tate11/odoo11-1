Denker - Orden de Compra

Grupo Denker, www.grupodenker.com

  •    Módulo para agregar las órdenes de compra al pedido de venta y adjuntarlas al correo de la factura.

Éste módulo agrega los siguientes campos para adjuntar la orden de compra

[res.partner] - Partner:
[dnk_purchase_order_required] - Is Purchase Order File required?
[dnk_attach_purchase_order] - Attach Purchase Order File?

[sale.order] - Orders:
[dnk_purchase_order_name] - Purchase Order Name
[dnk_purchase_order_file] - Purchase Order File

[account.invoice] - Invoices:
[dnk_purchase_order_name] - Purchase Order Name (Related)
[dnk_purchase_order_file] - Purchase Order File (Related)
